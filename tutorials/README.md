## Need Help Choosing?

**New to barcodes?** Start with [Read Barcodes from Images](./read-barcodes-images.md) to understand the one-line API, then move to [Generate Your First Barcode](./generate-barcode-basic.md) to create your own.

**Building a production system?** Review [Batch Processing with Multi-Threading](./batch-processing.md) and [ASP.NET Core Integration](./aspnet-core-integration.md) for scalable implementations.

**Working with PDFs?** Check [Read Barcodes from PDFs](./read-barcodes-pdfs.md) for native PDF processing without image extraction.
